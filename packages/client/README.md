# nftstorage.link client utilities

Utilities for working with the NFT.Storage IPFS Edge Gateway.

## Install

```console
npm install nftstorage.link
```

## Libraries and utilities

- [perma-cache](./docs/perma-cache.md)
- [gateway](./docs/gateway.md)
